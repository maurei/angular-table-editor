HTML5 Date and Disabled Cells

A bit of common configuration

An example demonstrating how certain fields can be made static while others remain editable, and showing how to use the supported HTML5 date type. The additional directives used are te-cell-active and te-cell-options. In this example, the third row is picked out by programmatically setting te-cell-active to false by checking for row.givenName != 'fleur', and the entire Company column is disabled by hardcoding the same directive to false. For the HTML5 Date support only works when providing ng-model with a Date object, as shown in the JavaScript.
